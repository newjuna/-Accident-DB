v22 원본DB 링크 및 산재 승인 요약 이미지 보정본

이번 수정 내용
1. 사이드바에 '사고 원본DB' 링크 추가
   - 구글시트 원본DB로 새 창 이동
   - 팀원들이 대시보드에서 바로 원본 사고 데이터를 확인할 수 있도록 구성

2. 산재 승인 사고 대시보드의 요약 생성 이미지 수정
   - 제목: 'YYYY년 M월 산업재해 현황'
   - 부제: '기준 : YYYY.MM.01 ~ YYYY.MM.DD'
   - 'KPI 집계대상' 표현 제거
   - '3대유형'으로 변경하고 하단에 '넘어짐·무리한 동작·물체에 맞음' 표기
   - 91일 이상 재해 건수는 선택한 해당 연도/월 기준으로 집계
   - 재해유형별 건수는 KPI분류가 아니라 실제 재해유형 기준으로 표시
   - 영업부별 근로손실일수 영역이 박스 밖으로 튀어나오지 않도록 재배치
   - 핵심 포인트에서 KPI 표현 제거

3. 기존 기능 유지
   - v21 로딩 진행률/게이지 유지
   - 출퇴근재해 제외 기준 유지
   - 산재승인 엑셀 색상 처리 유지
   - 91일 이상 재해 매장 페이지 이동 유지
