산재대시보드 v14 기준 - 반복사고 매장 팝업만 수정

기준
- 사용자가 다시 전달한 v14 파일을 기준으로 작업했습니다.
- Code.gs는 변경하지 않았습니다.
- index.html의 app.js?v=14는 유지했습니다.

수정 범위
- 반복사고 매장 화면의 '반복사고 발생 매장 리스트'에서 매장을 클릭했을 때 나오는 사고 리스트 팝업만 카드형 디자인으로 변경했습니다.

유지한 것
- 데이터 조회 화면 기능 및 디자인 유지
- 사고 상세보기 팝업 유지
- 반복사고 매장 메인 리스트 유지
- 대시보드 유지
- 기존 API 호출 구조 유지

적용 파일
- index.html
- app.js
- style.css
- logo.png

적용 방법
1. ZIP 압축을 풉니다.
