v23 원본DB 안전보건팀 전용 표시 및 산재 요약 우측 카드 보정본

이번 수정 내용
1. 사고 원본DB 링크를 안전보건팀 전용으로 변경
   - 수도권/지방 로그인에서는 표시되지 않음
   - 안전보건팀 로그인 시 '② 산재 승인 사고' 메뉴 안에서 확인 가능
   - 클릭 시 구글시트 원본DB가 새 창으로 열림

2. 산재 승인 사고 요약 생성 이미지 우측 영역 재정리
   - 기존 우측 큰 카드 안에서 항목이 어색하게 섞여 보이던 문제 보정
   - 우측 상단: 재해유형별 건수
   - 우측 하단: 영업부별 근로손실일수 + 91일 이상 재해 TOP
   - 박스 밖으로 내용이 튀어나오지 않도록 위치와 간격 재배치

3. 기존 기능 유지
   - v21 로딩 진행률/게이지 유지
   - 출퇴근재해 제외 기준 유지
   - 산재승인 엑셀 색상 처리 유지
   - 91일 이상 재해 매장 페이지 이동 유지
