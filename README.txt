v14 PC 안정 복구본

목적
- PC에서 안정적으로 동작하던 v14 기준으로 되돌린 파일입니다.
- Teams 자동 최신 접속용 start.html 방식은 제외했습니다.
- 모바일 하단 탭바/자동 캐시 우회 방식도 적용하지 않았습니다.
- PC 사용을 우선합니다.

반영 상태
- 데이터 조회: 매장명 검색창 처음부터 표시
- 데이터 조회: 연도/월 기준 영업부 드롭다운 제한 및 빠른 영업부 조회 유지
- 반복사고 매장: 기존 v14 기준 유지
- 반복사고 매장 사고 리스트 팝업 색상 보정 유지
- Code.gs 변경 없음

GitHub Pages 업로드 파일
- index.html
- app.js
- style.css
- logo.png

중요
- start.html은 업로드하지 마세요.
- Teams에는 /start.html 주소를 공유하지 마세요.
- PC 사용자는 기존 GitHub Pages 메인 주소로 접속하면 됩니다.

적용 방법
1. GitHub Pages 저장소에서 index.html / app.js / style.css / logo.png 덮어쓰기
2. 이전에 업로드한 start.html이 있다면 삭제
3. PC에서 Ctrl + F5로 새로고침
4. Teams 공유는 기존 메인 주소 사용
