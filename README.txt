v15 안전보건팀 산재승인 조회 최종 보정본

이번 수정 내용
1. 산재승인 조회 화면 엑셀 다운로드 추가
   - 조회된 결과 전체를 엑셀로 다운로드
   - 주민등록번호는 포함하지 않음

2. 불필요 필터 삭제
   - 산재승인 필터 삭제
   - KPI분류 필터 삭제
   - 남은 필터: 연도, 월, 영업부, 팀, 매장명 검색

3. 출퇴근재해 완전 제외
   - 산재승인DB에는 출퇴근재해가 있어도 됨
   - 대시보드/산재승인 조회/카드/그래프/요약 4페이지/엑셀 다운로드에는 출퇴근재해가 전혀 카운팅되지 않음
   - 기준: KPI집계현황분류에 '출퇴근'이 포함된 행은 제외

4. 상단 카드 구성 변경
   - KPI 집계대상
   - 업무상 사고 승인
   - KPI 제외
   - 총 근로손실일수
   ※ KPI 집계대상 카드 하단의 '(넘어짐·무리한 동작·물체에 맞음)' 문구는 유지

5. 3개년 비교 그래프 추가
   - 2024 / 2025 / 2026 KPI 반영 건수 비교
   - 세로막대그래프 + 붉은 점선 추세선
   - 불필요한 범례, 눈금선, 축 제거

6. 불필요 안내문구 제거
   - '안전보건팀 전용 화면입니다...'
   - '산재승인 유무 Y 기준'
   - '요양시작일~종료일 기준'
   - '원본DB AC~AE 기준...'
   위 문구는 제거 또는 정리

7. 요약 생성 4페이지 보정
   - 기존 1~3페이지 유지
   - 4페이지 산재승인/KPI 현황에서 출퇴근재해 제외
   - 4페이지에도 3개년 KPI 반영 건수 비교 그래프 반영
