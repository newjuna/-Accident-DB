v14 안전보건팀 통합 3장 요약 + KPI 비교기준 표기 보정본

이번 수정
1. 상단 KPI 카드 비교문구에 기준값 표시
   - 총 재해 카드: 전월 n건 대비 / 동일 또는 증감
   - 연간 누적 카드: 전년 동기 n건 대비 / 증감률

2. 예시 표기
   - 총 재해 22건 / 전월 22건 대비 동일
   - 총 재해 10건 / 전월 22건 대비 ▼ 12건
   - 연간 누적 77건 / 전년 동기 45건 대비 ▲ 71.1% 증가

3. 색상 기준 유지
   - 증가: 빨강
   - 감소/하락: 파랑
   - 동일: 초록

4. 기존 기능 유지
   - 수도권/지방 로그인
   - 안전보건팀 통합 로그인
   - 안전보건팀 3장 ZIP 다운로드
   - 데이터 조회, 엑셀 다운로드, 반복사고 기능 유지
