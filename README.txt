v20 산재 승인 사고 대시보드/엑셀/로딩 보정본

이번 수정 내용
1. 산재승인사고 대시보드의 '91일 이상 재해 매장' 카드에 페이지 이동 버튼 추가
   - 한 번에 5개씩 표시
   - 좌/우 화살표로 다음 5개 확인 가능
   - 항목 클릭 시 사고 상세보기 연결 유지

2. 산재승인사고 데이터 조회 엑셀 다운로드 스타일 보정
   - H열(근로손실일수)이 91일 이상인 셀:
     검정 채우기(#000000), 맑은고딕 굵게, 12pt, 흰색 글씨
   - I열(KPI집계현황분류)이 KPI 제외가 아닌 3대 재해 유형인 셀:
     초록 채우기(#00B050), 맑은고딕 굵게, 12pt, 흰색 글씨

3. 산재 승인 사고 화면 로딩 최적화
   - 같은 연도/월/검색 조건은 캐시 데이터 재사용
   - 대시보드/데이터조회/중상해매장 이동 시 이미 불러온 조건이면 로딩 없이 즉시 화면 전환
   - 처음 조회하거나 연도/월/검색 조건이 바뀌어 서버 조회가 필요할 때만 로딩 표시

4. 기존 기준 유지
   - 출퇴근재해 제외
   - 3대 재해 카드 색상 주황색 유지
   - 사고경위서 사고 영역 기존 기능 유지
