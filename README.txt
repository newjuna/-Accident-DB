v14 요약 생성 SVG 고정본

이번 수정의 핵심
- 기존 html2canvas가 DOM 레이아웃을 잘못 잡아 큰 공백/하단 잘림이 발생했습니다.
- 이번 버전은 요약 이미지를 HTML/CSS 레이아웃으로 캡처하지 않고,
  1280x720 SVG 보고서 이미지를 직접 생성한 뒤 PNG로 변환합니다.
- 따라서 미리보기 화면과 다운로드 이미지가 동일하게 나옵니다.

반영 내용
1. 요약 생성 버튼 유지
2. 우측 상단 logo.png 사용
3. 제목: YYYY년 M월 산업재해 현황
4. KPI: 총 재해 / 전월 대비 / 연간 누적
5. 월별 재해 발생 추이
   - 선택월까지 실적
   - 이후 월은 전년도 동일월 기준 과거건수
   - 붉은 점선 추세선
6. 재해유형 비중 도넛
7. 재해유형 TOP 3 / 영업부별 재해 TOP 3 / 핵심 포인트
8. 다운로드 이미지는 1280x720 PNG로 저장
9. Code.gs 변경 없음

업로드 파일
- index.html
