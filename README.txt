v14 PC 안정본 + 스타일 적용 엑셀 다운로드

이번 반영 내용
1. 데이터 조회 엑셀 다운로드 양식 개선
   - A1:G1 제목 병합
   - A1 제목은 파일명과 동일
   - 제목 글꼴: 맑은 고딕, 굵게, 20pt
   - 표는 A4부터 시작
   - A4:G4 헤더 색상: #DCE6F1
   - A4:G마지막행 전체 테두리: #808080
   - 바깥쪽/안쪽 윤곽선 모두 적용
   - 첫 표 행 고정 및 필터 적용
   - 열 너비는 내용 길이에 맞춰 자동 산정

2. 엑셀 표기 항목
   - No
   - 재해일자
   - 영업부
   - 팀
   - 매장명
   - 재해유형
   - 사고내용

3. 개인정보 보호
   - 재해자명, 사번은 엑셀에 포함하지 않음

4. 기존 기능 유지
   - PC 안정본 유지
   - 반복사고 매장 리스트 10개 단위 페이지 처리 유지
   - Code.gs 변경 없음
