v14 안전보건팀 통합 로그인 + 3장 요약 생성 확장본

반영 내용
1. 로그인 선택 항목 추가
   - 수도권영업부문
   - 지방영업부문
   - 안전보건팀

2. Code.gs 수정
   - 안전보건팀 통합 조회용 상수 추가
   - 안전보건팀 로그인 비밀번호 기본값: 9999
   - 안전보건팀으로 조회하면 수도권+지방 원본 데이터를 합산하여 반환
   - 기존 수도권/지방 로그인과 기존 기능은 유지

3. 요약 생성 기능
   - 수도권영업부문 로그인: 기존처럼 수도권 1장 생성
   - 지방영업부문 로그인: 기존처럼 지방 1장 생성
   - 안전보건팀 로그인: 3장 생성
     1페이지: 수도권영업부문 요약
     2페이지: 지방영업부문 요약
     3페이지: 전체 통합(수도권+지방) 요약 + 다음달 중점관리 포인트

4. 다운로드 방식
   - 일반 영업부문 로그인: PNG 1장 다운로드
   - 안전보건팀 로그인: PNG 3장을 ZIP 파일로 묶어 다운로드
   - JSZip 로딩 실패 시 PNG 3장을 순차 다운로드하는 fallback 포함

적용 파일
- index.html
- app.js
- style.css
- logo.png
- Code.gs

중요
- 이번에는 Code.gs도 수정됨.
- Apps Script에 Code.gs를 붙여넣은 뒤 반드시 새 버전으로 배포해야 함.
- 배포 URL이 바뀌면 app.js의 API_URL도 다시 확인해야 함.
- 기존 디자인과 데이터 조회/엑셀/반복사고 기능은 최대한 건드리지 않았음.
