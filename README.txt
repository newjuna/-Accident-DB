v18 산재 승인 사고 대시보드 레이아웃/캐시 보정본

이번 수정 내용
1. 산재 승인 사고 대시보드의 연도/월/요약 생성 영역을 사고경위서 대시보드처럼 우측 상단으로 이동
2. 산재 승인 사고 대시보드 연도 드롭다운에 '전체' 선택 가능
   - 2024/2025/2026 전체 데이터를 한 번에 볼 수 있음
3. 영업부별 TOP5와 팀별 TOP5를 하나의 박스 안에서 좌/우로 분리 표시
4. 산재 승인 사고 대시보드 그래프 레이아웃을 사고경위서 대시보드와 더 유사하게 보정
5. 산재 승인 사고 메뉴 이동 시 불필요한 재조회/로딩 감소
   - 이미 불러온 데이터가 있으면 화면 전환 시 캐시 데이터를 재사용
6. 기존 출퇴근재해 제외 기준 유지
7. 기존 사고경위서 대시보드/데이터조회/반복사고 기능 유지
