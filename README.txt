반영 내용
1. 대시보드 버튼 문구 변경
   - '월례회의 1장 요약' → '요약 생성'

2. 요약 생성 화면 디자인 변경
   - 제목: 'YYYY년 M월 산업재해 현황'
   - 우측 상단에 logo.png 사용
   - 상단 KPI 3개: 총 재해 / 전월 대비 / 연간 누적
   - 좌측: 월별 재해 발생 추이
     · 선택월까지는 실적
     · 이후 월은 전년도 동일 월 데이터를 '과거건수'로 표시
     · 붉은 점선 추세선 추가
   - 우측: 재해유형 비중 도넛
   - 하단: 재해유형 TOP 3 / 영업부별 재해 TOP 3 / 핵심 포인트

3. 이미지 다운로드 방식 변경
   - 상단 제어영역(제목줄, 다운로드버튼, X버튼)과 하단 안내문구는 PNG에 포함되지 않음
   - 실제 보고서 본문(#summaryReportPage)만 저장

4. 기존 기능 유지
   - 데이터 조회, 엑셀 다운로드, 반복사고 페이지 기능 유지
   - Code.gs 변경 없음
