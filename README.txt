v24 원본DB 메뉴 분리 및 산재 요약 이미지 그래프 보정본

이번 수정 내용
1. 사고 원본DB 메뉴 분리
   - '② 산재 승인 사고' 메뉴 안이 아니라 별도 메뉴로 분리
   - 안전보건팀 로그인에서만 표시
   - 수도권/지방 로그인에서는 표시되지 않음

2. 산재 승인 사고 요약 생성 이미지 수정
   - 3개년 산업재해 승인 건수 비교 그래프에 '각 연도 n월 기준' 문구 추가
   - 그래프 하단에도 월 기준을 표시하여 제목의 월과 비교 기준이 헷갈리지 않게 보정
   - 91일 이상 재해 매장 TOP 영역을 우측 카드에서 제거
   - 91일 이상 산업재해 발생 매장은 핵심 포인트 하단에 부서명/팀명/매장명/건수/일수 형태로 표시

3. 우측 그래프 정리
   - 우측 상단: 재해유형별 건수
   - 우측 하단: 영업부별 근로손실일수
   - 기존보다 간격을 넓히고 박스별로 분리해 허접해 보이는 느낌을 줄임

4. 기존 기능 유지
   - 로딩 게이지
   - 출퇴근재해 제외
   - 산재승인 엑셀 색상 처리
   - 91일 이상 재해 매장 페이지 이동
