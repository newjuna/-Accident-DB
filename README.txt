산재대시보드 v14 요약 생성 레이아웃 보정본

수정 내용
1. 요약 생성 미리보기 화면이 모달 안에서 잘리지 않도록 축소 표시
2. 이미지 다운로드 시에는 미리보기 축소가 제거되고 원본 1280x720 보고서 본문만 저장
3. 상단 모달 제목줄, 이미지 다운로드 버튼, X 버튼, 하단 안내문구는 다운로드 이미지에 포함되지 않음
4. 도넛 차트 크기 축소
5. 월별 추이 그래프 높이 축소
6. 하단 TOP3 / 핵심 포인트 영역이 잘리지 않도록 높이 재배치
7. 총 재해 카드의 하단 중복 문구를 '선택 월 발생 건수'로 수정
8. 대시보드 버튼명은 '요약 생성' 유지
