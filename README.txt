산재대시보드 v14 데이터조회 필터개선 적용 파일

적용 방법
1. 이 ZIP을 압축 해제합니다.
2. index.html, app.js, style.css, logo.png 파일을 GitHub Pages 저장소에 덮어씁니다.
3. Code.gs는 기능 변경을 하지 않았지만, 확인용으로 같이 포함했습니다. Apps Script는 바꾸지 않아도 됩니다.
4. 배포 후 브라우저에서 Ctrl + F5로 강력 새로고침합니다.

주의
- 이번 ZIP은 한글 폴더명 없이 루트에 파일을 바로 넣었습니다.
- 기능은 데이터 조회 필터 개선만 반영했습니다.
- 기존 v13 디자인은 유지했습니다.
